# Rival Rush

Small browser-based 3D-style racing game built with plain HTML, CSS, and JavaScript.

For a step-by-step launch guide, see `START_GUIDE.md`.

## Files

- `index.html`: page shell and HUD
- `style.css`: visual theme and responsive layout
- `game.js`: track generation, AI rivals, rendering, and controls

## How to play

Open `index.html` in a browser, or serve the folder locally:

```bash
cd racing-game
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Controls

- `Arrow keys` or `WASD`: steer, accelerate, brake
- `R`: restart after finishing the race
