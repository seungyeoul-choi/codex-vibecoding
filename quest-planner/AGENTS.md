# AGENTS.md

## 프로젝트 개요

- 이 프로젝트는 사용자의 기분, 사용 가능 시간, 예산, 동행 여부를 바탕으로
  미니 코스를 추천하는 입문자용 React 앱입니다.
- 첫 버전은 반드시 클라이언트 전용으로 유지하고, mock data와 `localStorage`
  만 사용합니다.
- 초보자도 읽고 확장할 수 있을 정도로 코드베이스를 단순하게 유지합니다.

## 작업 규칙

- 수정하기 전에 기존 코드와 문서를 먼저 읽습니다.
- 큰 리팩터링보다 작고 명확한 변경을 우선합니다.
- 수동 파일 편집은 `apply_patch`를 사용합니다.
- 분명한 실익이 없으면 새 라이브러리를 추가하지 않습니다.
- 파일 구조는 얕고 따라가기 쉽게 유지합니다.
- 초기 버전에는 백엔드, 인증, 외부 API 연동을 도입하지 않습니다.

## UI 기준

- 추천 결과에는 예상 시간과 예상 비용을 함께 보여줍니다.
- UI는 모바일과 데스크톱에서 모두 깨지지 않아야 합니다.
- UI 변경 후에는 반드시 브라우저에서 확인한 뒤 완료 처리합니다.
- 일반적인 화면 크기에서 레이아웃이 무너지지 않는지 확인합니다.

## 데이터와 상태

- 추천은 mock data를 사용합니다.
- `localStorage`에는 가벼운 사용자 선호나 기록만 저장합니다.
- 추천 로직은 결정적이어야 하며 소스에서 쉽게 추적할 수 있어야 합니다.

## 품질 기준

- 추상화보다 읽기 쉬운 컴포넌트 구조를 우선합니다.
- 코드만으로 충분히 이해되는 부분에는 주석을 남기지 않습니다.
- 동작이 바뀌면 관련 테스트, fixture, 문서도 함께 갱신합니다.
- 헌법이나 프로젝트 규칙에서 벗어나는 의도적 예외는 명확히 보고합니다.
