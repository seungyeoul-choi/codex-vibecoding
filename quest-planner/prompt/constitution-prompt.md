프로젝트 설명:
- 사용자 기분, 사용 가능 시간, 예산 동행 여부를 선택하면 오늘의 미니 코스를 추천하는 React 앱입니다.
- 첫 버전은 외부 API, 로그인, 서버 없이 mock data와 localStrorage만 사용합니다.

- 추천 결과의 시간과 비용 표시
- 모바일과 데스크톱에서 깨지지 않는 UI
- 구현 후 브라우저 확인
- 새 라이브러리를 최소화하는 기준
