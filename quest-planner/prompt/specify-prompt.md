Weekend Quest Planner의 첫 번째 MVP 기능을 명세로 작성해주세요.

목표:
사용자가 오늘의 기분, 사용 가능 시간, 예산, 동행 여부를 선택하면
조건에 맞는 미니 코스를 추천하는 앱을 만듭니다.

사용자:
- 주말이나 퇴근 후에 무엇을 할지 정하지 못한 사람
- 돈과 시간을 많이 쓰지 않고 가벼운 활동을 찾고 싶은 사람
- 복잡한 일정 앱이 아니라 바로 실행 가능한 추천을 원하는 사람

MVP 범위:
- React 기반 단일 페이지 앱
- 외부 API, 로그인, 서버 없음
- mock data 기반 추천
- 최근 추천 결과는 localStorage에 저장
- 모바일과 데스크톱 모두 지원

첫 기능:
- 기분, 시간, 예산, 동행 여부 입력
- 조건에 맞는 코스 추천
- 각 활동의 예상 시간과 예상 비용 표시
- 조건에 맞는 추천이 없을 때 빈 상태 안내

아직 기술 구현 계획은 작성하지 말고,
사용자 요구사항과 수락 기준 중심으로 명세를 작성해주세요.
