$Weekend Quest Planner는 React + Vite 기반 단일 페이지 앱으로 구현합니다.

기술 선택:
- React
- CSS는 일반 CSS 또는 기존 프로젝트 스타일 방식 사용
- 외부 API 없음
- 로그인 없음
- 서버 없음
- mock data는 프로젝트 내부 파일로 관리
- 최근 추천 결과 최대 3개는 localStorage에 저장
- 새 라이브러리는 설치하지 않음

구현 방향:
- 입력 폼 컴포넌트
- 추천 결과 카드 컴포넌트
- 최근 추천 결과 섹션
- 추천 계산 helper 함수
- mock activity data
- 빈 상태와 조건 완화 안내
- 모바일/데스크톱 반응형 확인

입문자가 이해하기 쉬운 파일 구조로 계획해주세요.
